∞ Infinity Graphics

A fully operational web design interface combining every major graphics rendering mode:

  • 🎨 2D Canvas — raster painting and photo editing
  • ✏️ SVG Vector — resolution-independent illustration
  • 🌐 WebGL 3D — real-time 3D rendering and visualisation
  • 🤖 AI Assist — smart color palettes, auto-enhance, and design suggestions

Built with pure HTML5, CSS3 and JavaScript — no external dependencies.
